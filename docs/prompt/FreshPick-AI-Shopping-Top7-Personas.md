## 0. 리서치 요약: 동네 마트 AI장보기 사용자 환경

### 0.1 구리시 인창동 지역 특성

구리시는 경기도 동부에 위치한 인구 약 20만 명의 중소 도시로, 서울 광진구·중랑구와 인접해 있어 서울 출퇴근 인구 비중이 높습니다. 인창동은 구리시의 주요 주거 지역으로, 1990~2000년대 개발된 아파트 단지와 구 시가지 주택가가 혼재합니다.

- **가구 구성**: 3~4인 가족 가구가 여전히 다수이나, 전국 트렌드와 마찬가지로 1인 가구 비중이 빠르게 증가 중 (전국 1인 가구 비중 약 35%)
- **연령 분포**: 30~50대가 주거 인구의 핵심. 고령 인구(65세 이상)도 꾸준히 증가
- **소득 수준**: 중위소득 가구가 주류 (월 가구소득 400~600만 원대). 서울 접경 베드타운 특성상 맞벌이 가구 비중 높음
- **상권**: 대형마트(이마트, 롯데마트)와 전통시장(구리전통시장), 동네 중소형 마트가 공존. FreshPick은 동네 마트가 타겟

### 0.2 온라인 식료품 구매 트렌드 (2025~2026)

오픈서베이 리포트와 푸드아이콘 분석을 종합하면:

- **온라인 구매율 85.7%**, 월평균 4.53회 온라인 구매. 오프라인(6.72회)과 병행 사용
- **40~50대의 온라인 이용 급성장**: 50대가 모든 연령대 중 온라인 구매 횟수 증가폭 1위
- **20대는 오히려 오프라인 회귀 경향** (직접 확인 후 구매 선호)
- **가격 민감도 상승**: 물가 체감으로 가격 비교·묶음 구매·특가 문구 영향력 증가
- **신선식품은 품질 우선**: 과일·채소·수산물은 가격보다 신선도·품질을 더 따짐
- **퀵커머스 인지도 68.9%**, 이용률 27.0%로 상승세 (특히 2~3인 가구)
- **오프라인 = 소량·낱개, 온라인 = 묶음·대량** 채널 역할 분화 정착
- **비계획형 구매 증가**: "갑자기 필요해서" 즉시성·접근성이 구매처 선택 핵심 요인

### 0.3 FreshPick AI장보기 서비스 컨텍스트

- **3탭 구조**: [AI채팅 · 내메모 · AI추천]
- **AI 어시스턴트**: "FreshPick" (고정 명칭)
- **5-tool Agent**: searchItems, getUserContext, getInventory, addToMemo, addToCart
- **5테마 AI추천**: 메뉴세트, 다시살때, 놓치지마세요, 다시발견, 신상품
- **customer_preference 기반**: 가구 인원, 식이 태그, 맛 태그, 알러지, 조리 실력, 선호 시간대

## 1. Top 7 페르소나

### 페르소나 시그니처 형식

각 페르소나는 v1.0의 `persona-signature.ts` 포맷과 대응됩니다:
```
gender:{M/F}|age_range:{Xs}|family:{N}|hour:{HH-HH}|diet:{tags}
```

### P1. 📌 "가족 저녁 매니저" — 김수진 (42세, 여성)

> *"오늘 저녁 뭐 해먹지? 아이가 안 먹는 건 빼고 추천해줘"*

**페르소나 시그니처**: `gender:F|age_range:40s|family:4|hour:15-18|diet:balanced`

#### 인구통계
| 항목 | 값 |
|------|-----|
| 연령 | 42세 |
| 성별 | 여성 |
| 거주지 | 구리시 인창동 (아파트 단지) |
| 가구 | 4인 (남편 45세 자영업, 초등 아들 10세, 중학 딸 14세) |
| 소득 | 월 가구소득 550만 원 (남편 자영업 + 본인 파트타임) |
| 학력 | 대졸 |

#### 생활 패턴
- **장보기 주기**: 주 3~4회 (동네 마트 + 주말 대형마트 1회)
- **쇼핑 시간대**: 평일 오후 3~5시 (아이 하교 전후)
- **월 식료품 지출**: 약 80~100만 원
- **조리 실력**: 중급 (한식 중심, 주 5회 이상 직접 조리)
- **주 구매 품목**: 정육, 채소, 계란, 유제품, 간식류

#### 온라인 소비 패턴
- **앱 사용 숙련도**: 중 (카카오톡·쿠팡·네이버쇼핑 익숙)
- **온라인 식료품 경험**: 쿠팡 로켓프레시 + 동네 마트 앱 병행
- **구매 결정 기준**: 1) 아이들이 좋아하는지 2) 신선도 3) 가격
- **AI 활용 기대**: 매일 반복되는 "저녁 메뉴 고민" 해결, 식재료 조합 추천

#### FreshPick AI장보기 사용 시나리오
1. 오후 4시경 "오늘 저녁 뭐 해먹지? 아이가 해산물 알러지 있어" → FreshPick이 알러지 가드 적용 후 메뉴 3개 추천
2. "삼겹살 재고 있어?" → getInventory 호출 → 재고 확인 후 수량 제안
3. "저번에 잘 먹었던 그 반찬 레시피 뭐였지?" → searchItems로 이전 주문 연관 레시피 검색
4. 추천받은 재료 "메모해줘" → addToMemo 호출

#### 디자인 시사점
- **UI 우선순위**: 메뉴 추천 카드 (사진 + 조리시간 + 난이도), 알러지 뱃지 필수 표시
- **톤앤매너**: 따뜻하고 실용적 ("오늘 이 메뉴 어때요? 아이들도 좋아할 거예요 😊")
- **핵심 기능**: 알러지 가드, 가족 인원 맞춤 수량 계산, 이전 주문 기반 재구매 추천

### P2. 📌 "효율 1인식" — 박준혁 (31세, 남성)

> *"오늘 뭐 먹지? 중식이 땡기네. 간단하고 칼로리 낮은 거"*

| 항목 | 값 |
|------|-----|
| 연령 | 31세 |

| 소득 | 월 350만 원 (IT 회사 직장인) |
| 학력 | 대졸 |

#### 생활 패턴
- **장보기 주기**: 주 1~2회 (주로 토·일 오후)
- **쇼핑 시간대**: 퇴근 후 저녁 6~8시 or 주말 오후
- **월 식료품 지출**: 약 30~40만 원 (외식·배달 별도 20만 원)

- **주 구매 품목**: 닭가슴살, 샐러드 채소, 계란, 두부, 간편식

#### 온라인 소비 패턴
- **앱 사용 숙련도**: 상 (IT 직장인, 새 앱 수용도 높음)
- **온라인 식료품 경험**: 쿠팡 헤비유저, 컬리 가끔
- **구매 결정 기준**: 1) 칼로리·영양 2) 조리 간편성 3) 가성비
- **AI 활용 기대**: "뭐 먹지" 결정 피로 해소, 다이어트 맞춤 추천

#### FreshPick AI장보기 사용 시나리오
1. 토요일 오후 5시 "중식이 땡기는데 칼로리 낮은 거 추천해줘" → 새우볶음밥+짬뽕국물 등 저칼로리 중식 메뉴 추천
2. "데친 고사리가 먹고 싶은데 뭐가 좋을까?" → searchItems로 고사리 활용 레시피 + 페어링 식재료 추천
3. 추천 메뉴의 재료 "장바구니에 담아줘" → addToCart (1인분 수량으로 조정)

#### 디자인 시사점
- **UI 우선순위**: 칼로리 표시, 조리 시간(10분 이내 강조), 1인분 수량 자동 계산
- **톤앤매너**: 캐주얼하고 효율적 ("깔끔한 중식 한 끼, 이렇게 해보세요!")
- **핵심 기능**: 1인분 수량 최적화, 간편 조리법 안내, diet 태그 필터링

|------|-----|

| 거주지 | 구리시 인창동 (아파트) |
| 가구 | 3인 (5세 아들) |
| 소득 | 월 가구소득 750만 원 (맞벌이) |

#### 생활 패턴
- **장보기 주기**: 주 1회 (토요일 오전~점심에 몰아서)
- **쇼핑 시간대**: 토요일 오전 10시~오후 1시
- **월 식료품 지출**: 약 90~120만 원
- **조리 실력**: 중급 (밀프렙·대량 조리 후 소분 냉동)
- **주 구매 품목**: 대량 정육, 냉동 채소, 밀키트, 간편식, 유아 간식

- **온라인 식료품 경험**: 쿠팡 + 마켓컬리 정기배송 + 동네 마트
- **구매 결정 기준**: 1) 일주일 식단 커버 가능한 구색 2) 대량 가성비 3) 아이 먹기 적합한 것
- **AI 활용 기대**: 주간 식단 계획 + 원스톱 장바구니 생성

#### FreshPick AI장보기 사용 시나리오

2. "이것도 같이 담아줘" → addToCart로 식단 전체 재료 일괄 추가
3. "지난주에 남은 양파랑 당근으로 뭐 할 수 있어?" → 잔여 식재료 활용 레시피

#### 디자인 시사점
- **UI 우선순위**: 주간 식단표 뷰, 장바구니 총액 실시간 표시, 밀프렙 가이드
- **톤앤매너**: 체계적이고 신뢰감 ("이번 주 식단, 이렇게 준비하면 평일이 편해져요")
- **핵심 기능**: 주간 식단 생성, 중복 식재료 합산, 대량 구매 가성비 안내

- **월 식료품 지출**: 약 15~20만 원 (외식·편의점 포함 30만 원)
- **조리 실력**: 초급 (라면, 볶음밥, 계란찜 정도)
- **주 구매 품목**: 라면, 계란, 햇반, 소시지, 과일, 과자

#### 온라인 소비 패턴
- **앱 사용 숙련도**: 상 (디지털 네이티브, SNS 영향 큼)
- **온라인 식료품 경험**: 쿠팡 로켓와우, 네이버쇼핑 최저가 비교
- **구매 결정 기준**: 1) 가격(최저가) 2) 소분 가능 여부 3) SNS 인기 상품
- **AI 활용 기대**: 예산 내 식단 계획, 초간단 레시피, 할인 상품 추천

#### FreshPick AI장보기 사용 시나리오
1. "만원으로 3끼 해먹을 수 있는 장보기 리스트 만들어줘" → 예산 맞춤 식재료 + 초간단 레시피 3개
2. "지금 할인하는 거 뭐 있어?" → getInventory + 할인 상품 필터
3. "계란으로 할 수 있는 요리 뭐 있어?" → searchItems로 계란 활용 레시피 5개 추천

#### 디자인 시사점
- **UI 우선순위**: 가격 강조 (할인율·총액), 레시피 난이도 별점, 소분 옵션 표시
- **톤앤매너**: 친근하고 재미있게 ("만원의 행복 🍳 이렇게 해볼까요?")
- **핵심 기능**: 예산 설정 + 추천, 초간단 레시피 (조리시간 10분 이내), 할인 알림

## 5. UI/UX 디자인 원칙 (Top 7 페르소나 공통)

### 5.1 필수 UI 요소

1. **알러지 뱃지**: 모든 추천 카드에 알러지 정보 시각적 표시 (P1, P3, P4, P7 필수)
2. **조리시간 뱃지**: ⏱️ 아이콘 + 분 단위 표시 (P2, P5, P7 핵심)
3. **인원 수량 조절**: 슬라이더 or +/- 버튼 (P1 4인, P2 1인, P3 3인, P6 6인+ 대응)
4. **가격 정보**: 총액·1인당 단가·할인율 명확 표시 (P5 필수, 전체 보조)
5. **건강 태그**: 저염·저당·고단백·저칼로리 뱃지 (P2, P4 필수)

### 5.2 톤앤매너 가이드라인

- FreshPick 어시스턴트는 **"동네 마트의 친절한 장보기 친구"** 포지셔닝
- 연령대별 존칭 자동 조절: 20~30대 "~해 보세요", 50대+ "~하시면 좋겠어요"
- 이모지는 절제하되 핵심 포인트에 1~2개 사용
- 추천 사유는 항상 1~2문장으로 (왜 이 메뉴/식재료인지)

### 5.3 접근성 기준

- 최소 터치 타겟: 44×44px (시니어 P4 고려)
- 대비율: WCAG AA 이상
- 폰트 크기: 기본 14px, 시니어 모드 16px+
- 원핸드 조작: 핵심 CTA는 하단 thumb zone 배치 (P7 퇴근길 시나리오)
