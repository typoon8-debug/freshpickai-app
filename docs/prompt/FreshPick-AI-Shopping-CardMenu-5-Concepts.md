# FreshPick AI장보기 — 사회문화 트렌드 기반 "나만의 카드메뉴" 컨셉 5가지

> **작성일**: 2026-05-04 (v2 — 사회문화 트렌드 기반 리디자인)
> **대상 앱**: freshpick-app v1.0 AI장보기
> **핵심 전환**: 전통적 기능 뱃지(알러지·조리시간·인원수) 배제 → **건강 + 가격** 요소만 유지 + **사회문화 Hot Trend 기반 카드메뉴** 신규 도입
> **디자인 방향**: 사용자가 AI장보기를 통해 "나만의 카드메뉴"를 만들고, 사용하고, Decoration하는 Personalize 경험

---

## 0. 컨셉 전환 배경

### 기존 접근의 한계
알러지 뱃지, 조리시간, 인원수 같은 **기능 중심 뱃지**는 경쟁 앱(쿠팡, 컬리, SSG)에서도 동일하게 제공하는 Commodity 요소. FreshPick만의 차별화가 불가능.

### 새로운 방향: "나만의 카드메뉴" 시스템
- **건강**(저속노화·혈당관리·간헐적단식 등)과 **가격**(예산 설정·가성비)은 기본 인프라로 유지
- 그 위에 **사회문화 트렌드를 반영한 테마 카드메뉴**를 얹음
- 사용자가 카드를 **선택·커스터마이즈·공유·수집**하는 게이미피케이션 요소 도입
- 카드메뉴 = AI가 생성한 레시피+장보기 리스트+스토리가 합쳐진 하나의 "콘텐츠 상품"

### 리서치 기반 트렌드 요약

**흑백요리사 시즌2** (2025.12~2026.01): 넷플릭스에서 공개되어 글로벌 Top 10에 3주째 1위를 기록하며 전례 없는 미식 열풍을 일으킴. 시즌3 제작 확정. 출연 셰프들의 식당 예약 경쟁이 벌어질 만큼 셰프 개인 브랜드가 대중화됨.

**혼웰식 (혼자+웰빙+식사)**: 2026년 식문화 핵심 키워드로 선정. 원 보울(One-bowl), 원 디쉬(One-dish) 포맷의 건강한 혼밥이 새로운 식문화로 자리잡는 추세. 덮밥·솥밥 간편식이 SNS 인증 콘텐츠로 활발히 공유됨.

**저속노화 간편식**: 정희원 교수와 세븐일레븐 협업 '저속노화 간편식 5종'이 큰 인기. 혈당 스파이크 방지, 대체 탄수화물, 고단백 구성이 편의점에서도 프리미엄 건강식을 찾는 소비 트렌드를 견인.

**간헐적 단식 대중화**: OMAD(하루 한 끼) 트렌드가 MZ세대 사이에서 확산. 16:8 방식이 가장 보편적이며, "한 끼를 제대로 먹자"는 의식이 식단의 질적 향상으로 연결.

**K-드라마 레시피 열풍**: 이태원 클라쓰 짬뽕탕, 사랑의 불시착 김치찜, 오징어게임 달고나 등 K-드라마 속 요리를 직접 만들어보는 콘텐츠가 글로벌로 확산.

## 1. 카드메뉴 아이디어 5가지

---

### 카드 1. 🔥 "셰프스 테이블" — 스타 셰프 코칭 카드

> *"오늘은 최강록 셰프의 간편 한식으로 저녁을!"*

**컨셉**: 흑백요리사·한식대첩 등 요리 서바이벌에서 스타덤에 오른 셰프들의 시그니처 레시피를, 동네 마트 식재료로 구현할 수 있도록 AI가 "셰프의 코칭"을 재현하는 카드.

**카드 구조**:
- **카드 전면**: 셰프 프로필 일러스트 + 시그니처 메뉴명 + 난이도 별점
- **카드 후면**: AI가 해당 셰프 스타일로 코칭하는 레시피 + 동네 마트 대체 식재료 매핑
- **Personalize**: 사용자가 좋아하는 셰프를 "My Chef"로 등록하면, AI채팅에서 해당 셰프의 어투와 요리 철학으로 추천

**예시 카드 라인업**:
- `Chef Choi's 간편한식` — 최강록 스타일 40년 내공의 한식 반찬 3종 세트
- `Chef Yoon's 전통주 안주` — 윤나라 스타일 전통주와 어울리는 안주 페어링
- `Chef Park's 스트리트 퓨전` — 흑수저 출신 셰프의 포장마차 감성 요리
- **시즌 한정 카드**: 흑백요리사 시즌3 방영 기간 한정 "미션 재현" 카드

**건강·가격 연동**:
- 건강: 각 셰프 레시피를 저염/저당 버전으로 AI가 자동 변환 옵션 제공
- 가격: "이 레시피, 우리 마트에서 ₩12,400이면 됩니다" 실시간 가격 매핑

**확장성**:
- 로컬 셰프 초대: 구리시 지역 맛집 셰프와 협업 → "인창동 맛집 셰프의 비밀 레시피" 카드
- UGC: 사용자가 만든 레시피를 "My Chef Card"로 발행 → 커뮤니티 공유
- 콜라보: 셰프 밀키트 상품과 연동 → 카드 클릭 시 해당 밀키트 바로 장바구니

**대응 페르소나**: P6 프리미엄 미식가(코스 요리), P2 효율 1인식(간편 버전), P1 가족 매니저(가족 버전)

### 카드 2. 🍚 "하루한끼 One Meal" — 간헐적 단식 완벽 한 끼 카드

> *"오늘의 단 한 끼, 영양 밸런스까지 완벽하게"*

**컨셉**: 간헐적 단식(16:8, OMAD)을 실천하는 사용자를 위해, "한 끼에 하루 필요 영양을 담는" 완전식을 AI가 설계하는 카드. 단순 칼로리 제한이 아닌, 혈당 안정 + 단백질 충분 + 포만감 유지의 과학적 한 끼.

**카드 구조**:
- **카드 전면**: 단식 타입 뱃지 (16:8 / 5:2 / OMAD) + 오늘의 메뉴 사진 + 영양 스코어
- **카드 후면**: 단백질·탄수화물·지방 비율 바 + 혈당 영향 지수 + 포만감 지속 시간 예측
- **Personalize**: 사용자의 단식 패턴(아침 스킵/저녁 스킵/주말 집중)에 맞춤

**예시 카드 시리즈**:
- `OMAD Power Bowl` — 닭가슴살+퀴노아+아보카도+견과류 원볼 세트
- `16:8 런치 세트` — 점심 한 끼에 집중하는 고단백 저GI 메뉴
- `Weekend Feast` — 5:2 방식의 "먹는 날" 프리미엄 한 끼
- `야간 보상식` — 야근 후 건강한 보상 한 끼 (혈당 스파이크 방지)

**건강·가격 연동**:
- 건강: 저속노화 원칙(혈당 스파이크 방지, 대체 탄수화물) 자동 적용
- 가격: "오늘의 완벽한 한 끼 = ₩8,500" 예산 최적화

**확장성**:
- 주간 단식 캘린더: 월~일 단식/식사일 플래닝 → 식사일 카드 자동 생성
- 커뮤니티: "나의 OMAD 기록" 공유 → 식단 인증 SNS 연동
- 연동: 헬스케어 앱(삼성 헬스, 애플 건강)과 체중·혈당 데이터 연동 (v1.x)

**대응 페르소나**: P2 효율 1인식(다이어트), P5 가성비 대학생(예산 한 끼), P7 워킹맘(시간 제한 한 끼)

### 카드 3. 🏠 "엄마손맛 가정식" — 세대를 잇는 레시피 아카이브 카드

> *"우리 엄마가 해주던 그 맛, AI가 복원합니다"*

**컨셉**: "엄마 손맛"이라는 한국 고유의 감성을 디지털로 아카이빙하는 카드. 사용자가 기억하는 가정식 레시피를 AI와 대화하며 복원하고, 카드로 소장·공유하는 감성 컬렉션.

**카드 구조**:
- **카드 전면**: 따뜻한 일러스트 (한식 밥상 느낌) + 요리 이름 + "○○네 가정식" 타이틀
- **카드 후면**: 복원된 레시피 + "이 맛의 기억" 사용자 메모 + 계절/절기 태그
- **Personalize**: 사용자 가족 고유 레시피를 AI가 인터뷰 형식으로 수집 → 카드화

**예시 카드 시리즈**:
- `엄마의 김치찌개` — "고기를 먼저 볶고 김치를 넣는 우리 집 방식" 복원
- `할머니의 명절 잡채` — "당면을 따로 볶는 3단 분리 방식" 가족 레시피
- `아빠의 주말 볶음밥` — 냉장고 잔반 활용 아빠표 볶음밥
- `절기 시리즈` — 입춘 나물, 동지 팥죽, 정월대보름 오곡밥 등 24절기 카드

**건강·가격 연동**:
- 건강: 원본 레시피 존중하되, "엄마 손맛 + 건강 업그레이드" 옵션 (나트륨 30% 감량 등)
- 가격: 제철 식재료 기반으로 가격 자동 최적화 ("봄 나물 세트 ₩9,800")

**확장성**:
- 레시피 상속: 부모님에게 AI가 전화/카카오로 레시피 인터뷰 → 자동 카드 생성
- 커뮤니티: "우리 동네 가정식 공유" → 인창동 이웃의 가정식 레시피 교환
- 시즌 이벤트: 명절 전 "우리 집 명절 상차림 카드" 자동 생성 → 장보기 리스트 연동
- 감성 디자인: 카드 자체가 "가족 요리책"처럼 수집·인쇄 가능한 디지털 굿즈

**대응 페르소나**: P1 가족 매니저(전승), P4 건강 시니어(기록자), P3 맞벌이 부부(간편 재현)

### 카드 4. 📺 "드라마 한 끼" — K-콘텐츠 레시피 카드

> *"어제 본 드라마 속 그 요리, 오늘 내가 만든다"*

**컨셉**: K-드라마·예능·영화에 등장하는 음식을 동네 마트 식재료로 재현하는 카드. 콘텐츠 소비와 식재료 소비를 잇는 "보고→먹는" 경험. 넷플릭스·티빙 시청 트렌드와 실시간 연동.

**카드 구조**:
- **카드 전면**: 작품 키비주얼 스타일 일러스트 + 메뉴명 + 작품 타이틀
- **카드 후면**: 원작 요리 vs FreshPick 간편 버전 비교 + 관련 에피소드 설명 + 장보기 리스트
- **Personalize**: 시청 기록 연동(v1.x), 좋아하는 장르별 카드 큐레이션

**예시 카드 시리즈**:
- `흑백요리사 미션 재현` — "3만원으로 코스 요리" 미션을 동네 마트 버전으로
- `사랑의 불시착 김치찜` — 북한식 정통 김치찜 간편 재현
- `오징어게임 달고나 세트` — 디저트 체험 키트 장보기
- `나의 해방일지 혼술 안주` — 드라마 속 혼술 감성 안주 3종
- `주간 트렌딩` — 그 주 가장 화제인 작품의 음식 자동 카드 생성

**건강·가격 연동**:
- 건강: 원작이 고칼로리여도 "건강 리메이크 버전" 동시 제공
- 가격: "드라마에서는 5만 원, FreshPick에서는 ₩15,000" 가성비 강조

**확장성**:
- 실시간 트렌드: 넷플릭스 Top 10 연동 → 그 주 화제 드라마 카드 자동 생성
- SNS 연동: "드라마 한 끼 인증샷" → 인스타그램/틱톡 공유 최적화 포맷
- 콜라보: 넷플릭스 코리아, CJ ENM 등과 공식 레시피 콜라보 (v1.x)
- 시즌 한정: 드라마 방영 기간 한정 카드 → 수집 요소

**대응 페르소나**: P5 가성비 대학생(SNS 인증), P2 효율 1인식(혼밥 재현), P7 워킹맘(아이와 함께)

### 카드 5. 🌿 "혼웰(HonWell) 라이프" — 나만의 웰빙 루틴 카드

> *"혼자 먹어도 우아하게, 건강하게, 나답게"*

**컨셉**: 2026 트렌드 키워드 "혼웰식"을 카드 시스템으로 체계화. 단순 혼밥이 아닌 "나의 건강과 라이프스타일을 설계하는" 웰니스 루틴을 AI가 설계하고, 사용자가 카드로 기록·수집·진화시키는 시스템.

**카드 구조**:
- **카드 전면**: 원 보울 사진 + 웰니스 스코어 + 기분/컨디션 태그
- **카드 후면**: 영양 밸런스 차트 + "오늘의 한 줄 웰니스 팁" + 식재료 원산지
- **Personalize**: 사용자 컨디션 입력(피곤/스트레스/운동 후/피부 관리 등) → 맞춤 식단

**예시 카드 시리즈**:
- `Morning Glow Bowl` — 피부 관리 원 보울 (연어+아보카도+베리류)
- `After-Run Refuel` — 운동 후 회복식 (고단백 덮밥)
- `Stress-Off Dinner` — 스트레스 해소 식단 (트립토판 풍부 메뉴)
- `Sunday Reset` — 주말 밀프렙 + 디톡스 원 보울
- `Seasonal Detox` — 계절별 디톡스 원 보울 (봄 해독, 여름 수분, 가을 보양, 겨울 온기)

**건강·가격 연동**:
- 건강: 저속노화 원칙 기본 적용. 혈당·단백질·항산화 스코어 카드에 표시
- 가격: "오늘의 혼웰 원 보울 = ₩7,200" 1인분 최적 가격

**확장성**:
- 루틴 캘린더: 주간 혼웰 식단 → 카드 7장 세트로 자동 생성
- 밀프렙 가이드: "이번 주 혼웰 장보기 리스트" 일괄 장바구니
- 데코레이션: 카드 배경색·폰트·스티커를 사용자가 커스터마이즈
- 레벨 시스템: 혼웰 카드 누적 → "혼웰 마스터" 뱃지 → 할인 쿠폰 연동
- SNS 포맷: 인스타 사이즈로 카드 내보내기 → "나의 혼웰 루틴" 인증

**대응 페르소나**: P2 효율 1인식(코어), P5 가성비 대학생(예산 혼웰), P7 워킹맘(퇴근 후 혼웰)

## 2. 카드메뉴 시스템 공통 아키텍처

### 2.1 카드의 3가지 레이어

```
┌─────────────────────────────────────────┐
│  Layer 1: 테마 카드 (셰프/한끼/가정식...)  │  ← 사회문화 트렌드
├─────────────────────────────────────────┤
│  Layer 2: 건강 + 가격 인프라              │  ← 기본 유지
│  · 저속노화 스코어 / 혈당 영향 지수       │
│  · 실시간 가격 매핑 / 예산 최적화          │
├─────────────────────────────────────────┤
│  Layer 3: AI Agent (FreshPick)           │  ← RAG + 5-tool
│  · searchItems / getUserContext           │
│  · getInventory / addToMemo / addToCart   │
└─────────────────────────────────────────┘
```

### 2.2 카드 생성 플로우

1. **AI 자동 생성**: FreshPick이 트렌드·사용자 취향·재고 기반으로 매일 새 카드 추천
2. **사용자 요청 생성**: "이태원 클라쓰 짬뽕탕 카드 만들어줘" → AI가 레시피 리서치 → 카드 발행
3. **커뮤니티 생성**: 다른 사용자가 만든 카드를 "Save" → 내 컬렉션에 추가
4. **운영자 큐레이션**: manager-app에서 시즌/이벤트 카드 발행 (흑백요리사 시즌3 등)

### 2.3 Personalize & Decoration

- **카드 배경**: 10+ 배경 테마 (한지 텍스처, 모던 미니멀, 레트로 감성 등)
- **카드 프레임**: 골드/실버/브론즈 → 사용 횟수에 따라 자동 승급
- **스티커**: 계절 스티커, 이벤트 스티커 → 카드 위에 장식
- **내보내기**: 인스타그램 스토리 / 카카오톡 공유 / 프린트 가능

### 2.4 게이미피케이션

| 요소 | 설명 |
|------|------|
| **카드 수집** | 시즌/이벤트 한정 카드 → 컬렉션 완성 보상 |
| **레벨** | 카드 사용 횟수 → Lv.1 요리새싹 ~ Lv.10 마스터셰프 |
| **뱃지** | "셰프스 테이블 10회 달성", "혼웰 30일 연속" 등 |
| **보상** | 할인 쿠폰, 프리미엄 카드 해금, 마트 포인트 연동 |

## 3. 카드메뉴 ↔ FreshPick v1.0 기술 매핑

| 카드메뉴 기능 | v1.0 구현 방식 |
|-------------|-------------|
| 셰프 레시피 RAG | `tenant_item_ai_detail` chunk_type='chef_recipe' + persona_tags=['chef_choi'] |

| 카드 공유 | Open Graph 메타태그 + 카카오/인스타 공유 API |
